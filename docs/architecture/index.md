# Data Platform Architecture

High-level overview of the data platform.

---

## 🧭 Overview

This platform is built on GCP and follows modern data engineering principles:

- Ingestion from multiple sources
- Processing via Apache Beam / Dataflow
- Storage in Cloud Storage + BigQuery
- Analytics via Looker
- Consumption through BI tools and applications
